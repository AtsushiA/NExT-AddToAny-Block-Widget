## Description

このブロックは、AddToAnyプラグインのウィジェット機能をブロックエディタで使えるようにします。

### 主な機能

- AddToAnyの共有ボタンを任意の場所に配置可能
- ウィジェットと同じ機能をブロックとして利用
- ボタンのカスタマイズオプション（サイズ、スタイル）
- レスポンシブ対応

### 使い方

1. AddToAnyプラグインがインストール・有効化されていることを確認
2. ブロックエディタで「AddToAny Widget」ブロックを追加
3. インスペクターでボタンのスタイルとサイズを設定

### 注意事項

このブロックを使用するには、[AddToAnyプラグイン](https://ja.wordpress.org/plugins/add-to-any/)が有効化されている必要があります。

## Installation

1. `/wp-content/plugins/next-addtoany-block`ディレクトリにプラグインファイルをアップロード、またはWordPressのプラグイン画面から直接インストール
2. WordPressの「プラグイン」画面でプラグインを有効化
3. AddToAnyプラグインが有効化されていることを確認
4. ブロックエディタで「AddToAny Widget」ブロックを使用

## Frequently Asked Questions

### AddToAnyプラグインは必須ですか？

はい、このブロックはAddToAnyプラグインの機能を利用するため、AddToAnyプラグインがインストール・有効化されている必要があります。

### どのような共有ボタンが表示されますか？

AddToAnyプラグインの設定に基づいた共有ボタンが表示されます。ボタンの種類はAddToAnyの設定画面でカスタマイズできます。

### ブロックエディタ以外でも使えますか？

このブロックはブロックエディタ専用です。クラシックエディタをお使いの場合は、AddToAnyの標準ウィジェットをご利用ください。
